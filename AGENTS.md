# Agents — start here

- **`CLAUDE.md`** — operating contract (how you work: memory, scope, commits, tooling). Read first. Agent-writable.
- **`.agent/roadmap.md`** — plan + status. **`.agent/memory.md`** — lessons + decision log (sole memory store).
- **`.claude/tech/*.md`** — subsystem reference, loaded on demand (filenames are the index).
- **`/session-prompt`** bootstraps a session (continues the roadmap; pass a task to override). **`/codex-review`** runs an adversarial Codex pass.

`README.md` is the human-facing project description.
